# Arduino Projects Portfolio

## Overview
This repository contains a collection of Arduino projects developed for learning, experimentation, and practical electronics applications.  
Each project includes source code, documentation, and hardware requirements.

The projects focus on:
- Embedded systems fundamentals
- Digital and analog I/O
- Sensors and actuators
- Beginner to intermediate Arduino concepts
